BMI Calculator using React

A simple and interactive BMI (Body Mass Index) Calculator built using React.
This application allows users to enter their height and weight to calculate their BMI and determine their health category.

Features

Input height and weight
Calculates BMI instantly

Displays BMI category:

Underweight
Normal weight
Obese

Fast and responsive UI
Clean and simple design

Technologies Used

React
JavaScript (ES6+)
CSS
Vite (for fast development build)
